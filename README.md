# Live HTML Interpreter 🚀

A browser-based real-time code editor that interprets HTML, CSS, and JavaScript instantly without page reloads. Perfect for testing small code snippets and learning web development fundamentals.

**🎥 Video Demo:** https://youtu.be/TVZ48ORo0kw?si=bm_RPp7KdTeo76nJ

## ✨ Features

- **Real-time Rendering**: Type code and see output instantly on the same page
- **No Dependencies**: Built with vanilla HTML, CSS, and JavaScript - no frameworks needed
- **Split View**: Code editor and live preview side-by-side
- **Beginner Friendly**: Great tool for HTML/CSS/JS practice
- **Lightweight**: Single HTML file, runs directly in any modern browser

## 🛠️ Tech Stack

- **HTML5** - Structure and iframe for output rendering
- **CSS3** - Styling and responsive layout
- **JavaScript (ES6)** - DOM manipulation, event listeners, real-time code execution

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Shivanshi9522/live-html-interpreter.git
   ```
2. Open `index.html` in your browser
3. Start coding in the left panel and see live output on the right

No npm install, no build step. It just works.
